# TCPPipe
Create [TCP](http://en.wikipedia.org/wiki/Transmission_Control_Protocol) to TCP, TCP to [UDP](http://en.wikipedia.org/wiki/User_Datagram_Protocol), UDP to TCP and UDP to UDP pipes.
